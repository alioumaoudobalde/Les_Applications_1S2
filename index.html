Partie 1 : Comprendre "Exercice 1 : Reconnaître une application"

Le concept clé

Pour qu'une correspondance soit une application, deux conditions doivent être réunies pour tout élément \(x\) de l'ensemble de départ :

  1. Existence : L'image \(f(x)\) doit exister (pas de division par zéro, pas de racine carrée de nombre négatif).
  2. Appartenance : Le résultat \(f(x)\) doit obligatoirement tomber dans l'ensemble d'arrivée indiqué.

Analyse d'exemples de la fiche

Exemple 1
Cas : \(f: \mathbb{R} \to \mathbb{R}^+\), \(f(x) = -x^2\)

Problème : L'ensemble d'arrivée est \(\mathbb{R}^+\) (nombres positifs). Si on prend \(x = 1\), le calcul donne \(-1^2 = -1\). Ce résultat n'est pas dans l'ensemble d'arrivée.

Verdict : Ce n'est pas une application.
Exemple 2
Cas : \(f: \mathbb{R} \to \mathbb{R}^+\), \(f(x) = x^2 + 1\)

Analyse : Un carré est toujours positif ou nul. Donc \(x^2 + 1\) sera toujours \(\geq 1\), donc toujours positif. Le résultat atterrit bien dans \(\mathbb{R}^+\).

Verdict : C'est une application.
Exemple 3
Cas : \(f: \mathbb{R} \setminus \{1\} \to \mathbb{R} \setminus \{3\}\), \(f(x) = \frac{2x+1}{x-1}\)

Départ : On enlève 1 (\(\mathbb{R} \setminus \{1\}\)), donc on ne divise jamais par zéro. C'est bon.

Arrivée : L'ensemble interdit la valeur 3. Est-ce que \(f(x) = 3\) est possible ?

\(\frac{2x+1}{x-1} = 3 \Rightarrow 2x+1 = 3(x-1) \Rightarrow 2x+1 = 3x-3 \Rightarrow x = 4\)

Mais \(x = 4\) est dans l'ensemble de départ, donc la valeur 3 pourrait être atteinte. Attends, vérifions :

\(f(4) = \frac{2\times4+1}{4-1} = \frac{9}{3} = 3\)

Donc la fonction pourrait donner 3, mais 3 n'est pas dans l'ensemble d'arrivée. Ce n'est donc pas une application.

Verdict : Ce n'est pas une application.

Partie 2 : Le cœur du devoir "Injection, Surjection, Bijection"

C'est la partie théorique la plus importante de l'Exercice 2. Retiens ces "tests" mentaux :

1. Injection (Injective)

Définition simple : Deux \(x\) différents ne donnent jamais le même résultat. On ne "gaspille" pas les flèches vers la même cible.

La méthode de preuve : On suppose que \(f( x_1 ) = f( x_2 )\) et on doit prouver à la fin que \( x_1 = x_2 \ ).

Contre-exemple : La fonction carré \(f: \mathbb{R} \to \mathbb{R}\), \(f(x) = x^2\) n'est pas injective sur \(\mathbb{R}\), car \(f( 2 ) = f( -2 ) = 4\). Deux entrées différentes donnent la même sortie.

2. Surjection (Surjective)

Définition simple : Tous les éléments de l'ensemble d'arrivée sont atteints au moins une fois. "Personne ne reste seul sur le banc de touche".

La méthode de preuve : On pose l'équation \(f(x) = y\). Pour n'importe quel \(y\) de l'ensemble d'arrivée, on doit pouvoir trouver au moins un \(x\).

3. Bijection (Bijective)

Définition : C'est à la fois injectif et surjectif. Chaque élément de départ a un unique partenaire à l'arrivée, et vice-versa. C'est un "mariage parfait".

Intérêt : Si c'est bijectif, on peut faire "marche arrière" (Fonction réciproque \(f^{-1}\)).

Partie 3 : Aide détaillée sur l'Exercice 2 (Le cas complexe)

L'exercice 2, question 1 est un classique difficile. La fonction est :

\(f: \mathbb{R} \to \mathbb{R}, f(x) = \frac{x-1}{x+1}\)

a) Montrer que \(f(x) \in ]-1, 1[\)

Explique ainsi :

Le dénominateur est \(x+1\). On peut remarquer que pour tout \(x \in \mathbb{R}\), on a :

\(-1 < \frac{x-1}{x+1} < 1\)

Pour le montrer, on peut étudier le signe de \(f(x) - 1\) et \(f(x) + 1\) :

\(f(x) - 1 = \frac{x-1}{x+1} - 1 = \frac{x-1 - (x+1)}{x+1} = \frac{-2}{x+1}\)
\(f(x) + 1 = \frac{x-1}{x+1} + 1 = \frac{x-1 + (x+1)}{x+1} = \frac{2x}{x+1}\)

En étudiant les signes selon les valeurs de \(x\), on trouve bien que \(-1 < f(x) < 1\).

b) Surjective ?

L'énoncé dit que l'ensemble d'arrivée est \(\mathbb{R}\) (tous les nombres).

Or, on vient de prouver que le résultat est toujours coincé entre -1 et 1.

Peut-on obtenir le résultat 5 par exemple ? Non, car 5 n'est pas dans \(]-1, 1[\).

Donc elle n'est pas surjective sur \(\mathbb{R}\).

c) Injective ?

Pour vérifier l'injectivité, on suppose \(f(x_1) = f(x_2)\) :

\(\frac{x_1-1}{x_1+1} = \frac{x_2-1}{x_2+1}\)

En faisant le produit en croix :

\((x_1-1)(x_2+1) = (x_2-1)(x_1+1)\)
\(x_1x_2 + x_1 - x_2 - 1 = x_1x_2 + x_2 - x_1 - 1\)
\(x_1 - x_2 = x_2 - x_1\)
\(2x_1 = 2x_2 \Rightarrow x_1 = x_2\)
C'est injectif.

Partie 4 : Restriction et Réciproque (Exercice 3)

C'est le lien entre la théorie et la pratique.

Le principe

Une fonction comme \(f: \mathbb{R} \to \mathbb{R}, f(x) = x^2\) n'est pas bijective (pas injective). Mais si on restreint l'ensemble de départ aux positifs seulement (\(\mathbb{R}^+\)), elle devient bijective et on peut trouver sa réciproque (la racine carrée).

Pour l'exercice 3.1 :

\(f: \mathbb{R} \to \mathbb{R}, f(x) = x^2 + 2x - 3\)
  • Elle n'est pas bijective sur \(\mathbb{R}\) car par exemple \(f(0) = -3\) et \(f(-2) = -3\). Deux \(x\) différents donnent le même résultat.
  • Mais sur l'intervalle \([-1, +\infty[\), elle devient bijective.

Comment trouver la réciproque ?

La méthode mécanique à apprendre par cœur :

  1. Poser \(y = f(x)\).
  2. Tout faire pour isoler \(x\) (cela demandera souvent de résoudre une équation du second degré avec Delta).
  3. L'expression finale de \(x\) en fonction de \(y\) est la réciproque \(f^{-1}(y)\).
Exemple pour \(f(x) = 2x + 3\)

1. On pose \(y = 2x + 3\)

2. On isole \(x\) : \(2x = y - 3 \Rightarrow x = \frac{y-3}{2}\)

3. La réciproque est \(f^{-1}(y) = \frac{y-3}{2}\)

Points Clés à Retenir

1
Une application doit avoir une image pour tout x et cette image doit être dans l'ensemble d'arrivée.
2
Injection : deux x différents → deux images différentes.
3
Surjection : tout élément de l'arrivée est atteint.
4
Bijection = injection + surjection → existence d'une réciproque.
